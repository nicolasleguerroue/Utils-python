Documentation LTSpice


Le 11 juin 2020 par Nicolas LE GUERROUE (version 1.0 )
Langage : PYTHON ( 3.7 )

  • Présentation


La bibliothèque LTSstruct permet de générer des fichiers LTSpice en commandes Python mais également de mettre en place des structures prédéfinies.
L'unité de placement des composants est une valeur entière qui, unitairement, représente le coefficient commun de taille des composants LTSpice
Ainsi, une résistance par exemple mesure 96*16.
Lorsque le placement d'un composant est à une distance de 1, cela veut dire que la distance représente 1/6 de celle de la longueur d'une résistance
Le curseur se déplacant de 16 en 16, cette unité de mesure introduite par la bibliothèque permet de toujours bien placer les composants pour qu'ils soient
accessibles par le curseur

  • Classe LTSpice

Description

Le constructeur permet de créer une instance de la classe LTStruct

Paramètres

[String] Nom du fichier LTSpice à générer (l'extension '.asc' est obligatoire)

Exemple

 ltspice = LTStruct("Fichier_LTSPice.asc")

  • setAutomaticExtend

Description

Cette méthode permet de cascader les structures automatiquement pour éviter qu'elles dépassent les anciennes

Paramètres

[Boolean] Etat de l'extension

Retour

[Boolean] La nouvelle valeur de l'état

Exemple

 ltspice.setAutomaticExtend(True)

  • voltageDivider

Description

Cette méthode permet de générer un pont diviseur avec deux résistances en série

Paramètres

[String] Label d'entrée
[String] Label de sortie
[Float] Valeur de la résistance supérieure
[Float] Valeur de la résistance inférieure

Retour

[Boolean] False en cas de succès

Exemple

 ltspice.voltageDivider("ve", "vs",  1000, 1200)

  • passiveHighPass

Description

Cette méthode permet de générer un filtre passe-haut d'ordre 1 consitué d'un condensateur et d'une résistance en série

Paramètres

[String] Label d'entrée
[String] Label de sortie
[Float] Valeur du condensateur
[Float] Valeur de la résistance

Retour

[Boolean] False en cas de succès

Exemple

  • passiveLowPass

  • invertingAop

Description

Paramètres
