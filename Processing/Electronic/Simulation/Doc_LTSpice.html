Documentation LTSpice


Le 05 juin 2020 par Nicolas LE GUERROUE (version 1.0 )
Langage : PYTHON ( 3.7 )

  • Présentation


La bibliothèque LTSpice permet de générer des fichiers LTSpice en commandes Python.
L'unité de placement des composants est une valeur entière qui, unitairement, représente le coefficient commun de taille des composants LTSpice
Ainsi, une résistance par exemple mesure 96*16.
Lorsque le placement d'un composant est à une distance de 1, cela veut dire que la distance représente 1/6 de celle de la longueur d'une résistance
Le curseur se déplacant de 16 en 16, cette unité de mesure introduite par la bibliothèque permet de toujour bien placer les composants pour qu'ils soient
accessibles par le curseur

  • addNode

Description

Cette méthode permet d'ajouter un label dans le schéma. Ce label, placé à un noeud du circuit, permet de revenir sur ce noeud plus tard, pour rajouter une autre branche de circuit.
Lorsque aucune coordonnée n'est donnée, le label se place aux coordonnées courantes.

Paramètres

[String] Nom du label
[String] État de vérification : si True, le label devra être unique dans le circuit [defaut=False]
[Integer] Position du label en x [coordonnées standard de LTSpice] (argument optionnel)
[Integer] Position du label en y [coordonnées standard de LTSpice] (argument optionnel)

Retour

[Boolean] False

Exemple

 ltspice.addNode("Vin", True, 10, 10) #Place un label unique appelé "Vin" en (10,10)

  • getNode

Description

Cette méthode permet de récuperer les informations d'un label

Paramètres

[String] Nom du label

Retour

[List] Une liste ["name", checks_state, x, y] si le label est trouvé. Le cas échéant, une liste avec un label vide aux coordonnées d'origine (["", False, 0, 0])

Exemple

 node = ltspice.getNode("Vin")

  • addResistor

Description

Cette méthode permet d'ajouter une résistance dans le circuit.
Lorsque aucune coordonnée n'est donnée, la résistance se place aux coordonnées courantes.

Paramètres

[String] Nom de la résistance
[Integer] Orientation de la résistance [UP, DOWN, RIGHT, LEFT]
[String] Valeur de la résistance en ohms [les préfixes 'k', 'meg', etc, ne fonctionnent pas encore, implémentation future]
[Integer] Position de la résistance en x [coordonnées standard de LTSpice] (argument optionnel)
[Integer] Position de la résistance en y [coordonnées standard de LTSpice] (argument optionnel)

Retour

[Boolean] False

Exemple

 ltspice.addResistor(name="R1", teta=RIGHT, value="1200",x=5, y =12 ) #Place une résistance R1 de 1200 Ohms en (5,12)

  • addAop

Description

Cette méthode permet d'ajouter un amplificateur opérationnel dans le circuit
Lorsque aucune coordonnée n'est donnée, l'AOP se place au coordonnées courante

Paramètres

[String] Nom de l'AOP
[Integer] Orientation de l'AOP [UP, DOWN]
[Boolean] Etat mode Miroir de l'AOP [0, 1] -> implémentation du mode_miror=1 non effectué
[String] Entrée courante (Entrée nagative ou positive) de l'AOP [suite de la portion de circuit précédente] -> implémentation du link="-" non effectué
[String] Label d'alimentation positive. Si ce label existe, il viendra alimenter l'AOP
[String] Label d'alimentation négative (ou masse). Si ce label existe, il viendra alimenter l'AOP
[String] Label de l'entrée opposée (si link="+", le label viendra se rattacher à l'entrée "-")
[Integer] Position de l'inductance en x [coordonnées standard de LTSpice] (argument optionnel)
[Integer] Position de l'inductance en y [coordonnées standard de LTSpice] (argument optionnel)

Retour

[Boolean] False

Exemple

 ltspice.addAop(name="U1", teta=RIGHT, miror=0, link="+",vcc_label="+VCC", vee_label="-VCC", negative_label="E-", x=5, y =12 )

  • Code de démonstration

Description

Voici le code d'exemple associé à la bibliothèque

Paramètres

Aucun paramètre attendu

Retour

[Boolean] False

Exemple

 lt = LTSpice("Project.asc")
lt.displayInfos(True)
lt.setCursor(0, 0)
lt.addNode("V_wave", True, lt.x(), lt.y())
lt.wire(RIGHT, 2)
lt.addResistor("R1", RIGHT,"1000")
lt.wire(RIGHT, 2)
lt.addNode("N1")
lt.wire(RIGHT, 2)
lt.addCapacitor("C1", RIGHT, "1000")
lt.wire(RIGHT, 2)
lt.addNode("N2")
lt.wire(DOWN, 2)
lt.addResistor("R3", DOWN, "1000")
lt.wire(DOWN, 2)
lt.addGround(DOWN, 2)
lt.setCursorToNode(lt.getNode("N2"))
lt.wire(RIGHT, 2)
lt.addAop(name="A1", teta=DOWN, miror=0, link="+", vcc_label="+VCC", vee_label="-VCC", negative_label="E1")
lt.wire(lt.right(), 2)
lt.addNode("Vs")
lt.setCursorToNode(lt.getNode("E1"))
lt.addGround(LEFT, 2)
lt.addText("This file has been automatically generated", -10,25)
lt.addDirective(".tran 11", -10, 5)
lt.addWaveFileInput(filename="son.wav", name="V_wave", x=-10, y=15)
lt.addDoubleSupply(vcc=15.0, vee=15.0, name=["+VCC", "-VCC"], x=-10, y=30)
lt.exportFile()
